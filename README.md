# DarkShop
Dark Shop is a Magical online shop just like those of movies. This site was actually inspired from "Cashero" - the korean drama. This whole website is kind of a dream which everyone must had when they were very young- well it was my dream atleast.

# Working of Site
- On clicking the next arrow button on the home page, you will be taken to instruction page where all the instructions are stated carefully.
- One must read all the instructions to understamd the whole shop exchange system. 
- There are two arrows at the bottom now, one takes you back toi the home and the other takes you to proceed forward to the store.
- On store you are provided with 100 currency points at the start and you can use it in the following shopping for the superpowers.
- Do not worry if you are finished with the currency you can earn more by clicking on earn more option.
- After earning return to the store and click on next option to see the receipt of powers you bought and the sacrifices you gonna make to use it.
